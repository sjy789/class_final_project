# 智能物体放置助手设计说明

## 应用目标

方向 A 的目标是给定一张背景图和一个前景物体，判断物体放在哪里更自然，并输出候选位置、评分、推荐结果和合成图。

本项目实现为 Web 应用：

1. 用户输入背景图和前景图。
2. 系统提取或估计前景 mask。
3. 候选位置生成模块产生多组候选框。
4. 放置评分模型对每个候选位置输出 0-1 合理性分数。
5. 排序模块返回 Top-1/Top-3/Top-5。
6. 合成模块输出最终图像。
7. 遮挡敏感性分析生成解释热力图。

## 模块对应

基础模型小改动：

- 输入从单纯 RGB 合成图扩展为候选框、前景 mask 质量、局部背景纹理、颜色和亮度差异等特征。
- 输出从二分类扩展为连续分数和“推荐/可接受/不推荐”三档标签。

模型改造进阶：

- 增加候选位置排序逻辑，对多个候选位置统一评分并返回 Top-K 推荐。

多模型/工具串联：

- 前景 mask 处理工具。
- 候选框生成工具。
- PyTorch 放置评分模型。
- 颜色协调工具。
- 图像合成工具。

模型解释：

- 使用遮挡敏感性分析，对背景不同区域进行遮挡并重新评分，得到解释热力图。

复杂交互：

- 上传图片。
- 自动推荐。
- 手动拖拽。
- 调节大小。
- Top-K 切换。
- 热力图显示。
- 导出合成图。

## 当前实现特点

评分模型是一个轻量 CPU 推理模型，位于 `backend/app/model.py`。模型用 PyTorch 实现线性评分头，输入为经过改造的结构化图像特征，适合在本地 CPU 上稳定演示。

图像流水线位于 `backend/app/pipeline.py`，负责串联 mask、候选生成、评分、排序、热力图和输出保存。
